# Обновление товаров на Ozon
Cкрипт автоматически обновляет информацию о товарах (часах) в интернет-магазине продавца на платформе Ozon. Он скачивает актуальные данные с сайта поставщика и загружает их в личный кабинет продавца на Ozon.

## Что делает сркипт

### Загружает остатки товаров от поставщика
1. Скачивает файл с сайта [timeworld.ru](timeworld.ru/upload/files/ostatki.zip)
2. Извлекает из него информацию о наличии часов на складе поставщика

### Обновляет остатки на Ozon
1. Получает список всех товаров продавца на Ozon
2. Сопоставляет артикулы из файла поставщика с товарами в магазине
3. Устанавливает правильное количество единиц товара на складе:
4. Если у поставщика указано ">10" — устанавливает остаток 100 штук
5. Если указано "1" — устанавливает остаток 0
6. Если указано другое число — переносит его напрямую
7. Товары, которых нет в файле поставщика, получают остаток 0

### Обновляет цены на Ozon
1. Сопоставляет цены из файла поставщика с товарами в магазине
2. Преобразует цену из формата "5'990.00 руб." в "5990"
3. Загружает актуальные цены на площадку

# Обновление товаров на Яндекс Маркете

Скрипт для платформы Яндекс Маркет (бывший Яндекс.Маркет) обновляет остатки и цены сразу для двух моделей работы: FBS (Fulfillment by Seller) и DBS (Delivery by Seller).

## Что делает скрипт для Яндекс Маркета

### Загружает остатки товаров от поставщика
1. Использует функцию download_stock() из модуля seller (скачивает файл с сайта timeworld.ru)
2. Извлекает информацию о наличии часов на складе поставщика

### Обновляет остатки на Яндекс Маркете
1. Получает список всех товаров продавца в кампаниях FBS и DBS
2. Сопоставляет артикулы (Код) из файла поставщика с товарами в магазине
3. Устанавливает правильное количество единиц товара на складе:
   - Если у поставщика указано ">10" — устанавливает остаток 100 штук
   - Если указано "1" — устанавливает остаток 0 (нет в наличии)
   - Если указано другое число — переносит его напрямую
   - Товары, которых нет в файле поставщика, получают остаток 0
4. Для каждого остатка указывает:
   - Код товара (sku)
   - Склад (warehouseId) — отдельно для FBS и DBS
   - Количество (count)
   - Тип остатка (FIT — доступно для продажи)
   - Дату и время обновления в формате UTC

### Обновляет цены на Яндекс Маркете
1. Сопоставляет цены из файла поставщика с товарами в магазине
2. Преобразует цену из формата "5'990.00 руб." в число 5990
3. Указывает валюту RUR
4. Загружает актуальные цены на площадку
